Shows ACF field names directly in the admin editor interface for both Classic and Gutenberg editors.

== Description ==

ACF Show Field ID is a developer-friendly plugin that displays Advanced Custom Fields (ACF) field names directly in the WordPress admin interface. This plugin works seamlessly with both the Classic Editor and the new Gutenberg Block Editor.

**Key Features:**

* **Field Name Display**: Shows ACF field names next to their labels
* **Copy to Clipboard**: Click on any field name to copy it instantly
* **Gutenberg Support**: Full compatibility with the Block Editor
* **Performance Optimized**: Only loads on relevant admin pages
* **Beautiful UI**: Modern, clean design that integrates seamlessly
* **Responsive Design**: Works great on all screen sizes

This plugin is essential for developers working with ACF who need to quickly identify field names for their code.

**Requirements:**
* WordPress 5.0 or higher
* Advanced Custom Fields (ACF) plugin
* PHP 7.0 or higher

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/acf-show-field-id` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Make sure the Advanced Custom Fields plugin is installed and activated
4. Go to any ACF field group or post/page with ACF fields to see the field names displayed

== Frequently Asked Questions ==

= Does this plugin work with Gutenberg? =

Yes! The plugin has full support for the Gutenberg Block Editor and will display field names for ACF blocks and fields.

= Does this plugin work without ACF? =

No, this plugin requires the Advanced Custom Fields plugin to be installed and activated. If ACF is not active, you'll see an admin notice.

= Can I copy the field names? =

Yes! Simply click on any displayed field name to copy it to your clipboard.

= Does this affect my site's frontend? =

No, this plugin only works in the WordPress admin area and has no impact on your site's frontend performance or appearance.

= Is this plugin compatible with ACF PRO? =

Yes, the plugin works with both the free ACF plugin and ACF PRO.
